Semester 1 development project

    Overview
    In this project, we worked in a team to create a 5 minute vertical slice of a game that runs on android, called Quest. Our idea was to create a 2D adventure game where you cast magic by drawing symbols on the screen.

    My role in the Team
    My role was the main programmer, so I was in charge of most of the main systems. Pathfinding, the magic recognition system as well as all of the art setup (lights,particle systems and animation trees) were done by me.
